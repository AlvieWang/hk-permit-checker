# 港澳通行证材料自查系统

模拟深圳公安出入境办事窗口的智能对话系统，帮助用户自助核查办理港澳通行证所需材料。

## 功能特点

- 🎨 **官方风格界面**：模拟深圳公安出入境服务大厅的视觉风格
- 💬 **智能对话**：逐步引导用户完成材料自查
- ✅ **实时清单**：左侧边栏实时显示材料准备进度
- 📱 **响应式设计**：支持PC和移动端访问
- 🚀 **纯前端实现**：无需后端服务器，可直接部署到GitHub Pages

## 使用说明

1. 打开网页后，智能助手会通过对话逐步询问
2. 点击选项按钮或输入文字回答问题
3. 左侧边栏会实时更新材料清单和办事指南
4. 完成后会生成个人材料自查报告

## 技术栈

- HTML5
- CSS3 (Flexbox + 渐变 + 动画)
- 原生JavaScript (无框架依赖)

## 本地测试

直接用浏览器打开 `index.html` 即可运行。

## 部署到GitHub Pages

1. 创建GitHub仓库
2. 上传所有文件到仓库
3. 在仓库设置中启用GitHub Pages
4. 访问 `https://[用户名].github.io/[仓库名]/` 查看效果
